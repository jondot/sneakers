Sneakers

High Performance Background Jobs. On Ruby.

  • Performance.

    Combined multi-process and multi-threaded processing model maxes out all your cores.

  • Workers.

    Out of the box worker management, metrics, message handling and more, while being highly configurable and easily extensible.

  • RabbitMQ.

    We built on RabbitMQ — a robust, performant and stable queue broker that everyone loves.

It's as simple as 1-2-3.

1. Install
$ gem install sneakers
2. Make a Worker
class TitleScraper
  include Sneakers::Worker
  from_queue 'downloads'

  def work(msg)
    doc = Nokogiri::HTML(open(msg))
    worker_trace "FOUND <#{doc.css('title').text}>"
    ack!
  end
end
3. Run
$ foreman start
# or require 'sneakers/tasks', and:
# $ WORKERS=TitleScraper rake sneakers:run
